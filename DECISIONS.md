오늘 날짜 기준 결정 사항 및 미결 사항 기록.

---

## 확정

| 항목 | 결정 |
|------|------|
| 툴 성격 | 내부 의사결정용. 고객용 아님 |
| 데이터 소스 | 블룸버그 엑셀 export |
| 파일 구조 | 파일 1개, 시트 다수. S_/I_ prefix로 타입 구분 |
| 전처리 책임 | 사용자가 raw → processed 직접 변환 |
| 롤링 처리 | significant 기준 없음. 적용/미적용 두 버전 병렬 출력 |
| 스프레드 | nC2 전체 조합. 기준 시트는 사용자 선택 |
| UI | Streamlit. 예쁜 거 우선순위 낮음 |
| core 설계 | Streamlit 의존성 없음. 순수 함수 |
