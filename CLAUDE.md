# CLAUDE.md

## 프로젝트 개요

채권펀드 구성 전 내부 의사결정 지원 툴.
고객 제안서 작성 전 단계에서 시장 파악, 자산 선별, 수익률 시뮬레이션을 위한 고급 분석 도구.
고객용 아님. 정보 밀도 높은 게 우선, 예쁜 건 나중.

## 데이터 규칙

### 원본 엑셀 구조 (raw/)
- 파일 하나, 시트 여러 개
- 시트명 prefix로 타입 구분:
  - `S_XXX` → 특정 섹터 채권풀의 **일별** 수익률 (XXX는 자유)
  - `I_XXX` → 특정 섹터 내 발행사별 **만기별 현재** 수익률 (XXX는 자유)
- 시트명 뒤 이름은 코드가 의미 해석 안 함. 라벨로만 사용.

### 각 시트 내부 구조
```
~ (스킵할 행, 몇 번째인지는 코드에서 확인 후 수정)
날짜 | 헤더1 | 날짜 | 헤더2 | 날짜 | 헤더3 ...
값   | 값    | 값   | 값    | 값   | 값
```
- 날짜-헤더 쌍으로 반복되는 구조
- 날짜는 블룸버그가 통일해서 제공 (interpolation 처리됨)
- 헤더 이름은 블룸버그가 붙여주는 것 그대로 사용

### processed/ 스키마 (고정)
**S 타입:**
```
date | maturity | yield
```

**I 타입:**
```
date | issuer | maturity | yield
```
- maturity는 숫자(년) 단위로 통일 (전처리는 사용자 책임)
- 전처리는 사용자가 raw → processed 변환 시 직접 수행

## 입력 파라미터

### 상수 (constants.py, hardcoded)
- 금통위 일정 (연간 8회)
- 현금성자산 규제비율

### 사용자 입력 (Streamlit UI)
- 현재 기준금리
- 펀드 개시일
- 펀드 만기
- 설정액
- 레포차입비율

- 금통위 시나리오 (회차별 인상/동결/인하 및 bp)

## 핵심 로직

### 가중평균 레포조달비용
- 개시일 기준으로 금통위 일정과 교차
- 구간별 일수 계산
- 기준금리 × 구간 일수 가중평균
- 레포스프레드는 별도 가산

```
예시:
개시일 2025-03-01, 만기 6개월(182일), 현재금리 3.50%
  03-01~04-17: 47일 @ 3.50%
  04-17~05-29: 42일 @ 3.50% (동결)
  05-29~07-10: 42일 @ 3.25% (-25bp)
  07-10~08-30: 51일 @ 3.25% (동결)
가중평균 = (47×3.50 + 42×3.50 + 42×3.25 + 51×3.25) / 182 = 3.37%
```

### 수익률 계산
롤링 적용/미적용 두 버전 항상 병렬 계산. significant 기준 없음, 사용자가 판단.
- 미적용: 편입자산 YTM 그대로
- 적용: YTM + 듀레이션 × 롤링수익률

### 스프레드 분석
nC2 전체 조합 계산. 기준 시트는 사용자가 선택.

### 편입자산 선택
- 신용등급 상/하한 필터
- target maturity 기준 금리 높은 순 정렬
- 커브가 유리하면 만기 허용범위 내에서 더 긴 것 포함 (허용범위는 TODO)
